XGpgSigSign.hsc
----------------
Dieses Script bitte einbinden unter:
Einstellungen->Automatische Abläufe->Artikel/Postings->Verarbeitung->ausgehender Artikel
siehe auch Bild sign.jpg

XGpgSigVerify.hsc
-----------------
Dieses Script bitte einbinden unter:
Einstellungen->Automatische Abläufe->Artikel/Postings->Eingehend
siehe auch Bild verify.jpg

Die Pfade in den Scripten bitte anpassen.

Das Passwort und den Key für GNUPG in die XGpgSig.ini eintragen.
